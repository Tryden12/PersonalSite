Wine Notes

Introduction

WineNotes is an application that allows users to quickly and neatly jot down notes about a specific wine. Users can use the sort functionality to sort the notes in many ways, so that a note will never get lost in the shuffle.

I worked solo on this project while attending APSU in the Spring of 2022.

Planning

The planning involved with this app took all of 3 days. I researched how to use Room with SQL Lite, how to parse/display time from the SimpleDateFormat class, and how to use Dao's correctly. This simple app was built in a quick amount of time, so MVVM was not necessary. Only two activities present.

For Room, I received help in setting up the abstract class database file. This file contains a companion object that will check if a database matching the database name has been created. If it has been built, it will grab it. For the NoteDao file, I watched tutorials on how to implement this file correctly in my MainActivity.

Result

Reflection

WineNotes was one of those projects that opened my mind up to what is possible when working with coroutines, Room, and integrating SQL into an Android app. Room is incredibly fun to use, and coroutines became my friend rather than an enemy. I now look for ways to implement coroutines, rather than before I would look for ways to avoid it (not smart!). Coroutines are essential to building a user friendly, quick response app.

Looking back, I wish I could have spent more time developing this app to learn even more about how to make a fully functioning note taking app. There are many more functionalities I would like to add - such as swipe delete, character/font stying, and lists.

Hopefully I can come back to this project to learn more and develop a fully functioning note taking app. For now, my focus remains on other topics.
