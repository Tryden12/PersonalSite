Titans Tabs

Introduction

The overall goal of this project is to create simplified version of the official Tenenssee Titans app.

Thus far, I set the app up for success by using MVVM architecture. Currently, I have an api file, Retrofit, Repository, ViewModel, Models, a Dao, Room database, an adapter and two Fragments. The login and signup activities have successful validation for creating a user and signing in using Room.

Titans Tabs is an application that is an ongoing side project.

Planning

The planning involved with this app took all of 3 days initially, and is ongoing.

As a Titans fan, I wanted to create an app that captures the Tennessee Titans. I decided to use this app as an opportunity to showcase my abilities in designing similar to the official Tennesse Titans app, and provide my own spin on things as well such as the login/create user screens.

Future edits will invole the utilization of Retrofit to parse player information JSON from an API url, and present this in a RecyclerView. The user will then have the ability to click on a player from the RecyclerView to view the player's full profile.

Additional edits will invovle news, stories, schedule, and more.

Result

Reflection

Titan Tabs is application that excites me because I get to infuse my interests of development and football. I cannot wait to polish off this app, and present its full version. I anticipate finishing the edits mentioned above over the course of the next couple months.
