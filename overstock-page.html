Overstock

Introduction

The overall goal of this project was to develop a mock retail website from scratch using the software development lifecycle (SDLC).

The project consisted of a group of four including myself. Over the course of 3 months, we worked together to create a final product we were happy with showcasing to our professor.

View Overstock Website

DISCLAIMER: Mobile view was not adjusted during this project. It is intended for desktop view only.

Planning

Overstock was developed using the SDLC over the course of 12 weeks.

Planning: 2 Weeks

Analysis: 2 Weeks

Design: 4 Weeks

Implementation: 2 Weeks

Testing & Integration: 2 Weeks

Maintenance: Ongoing

We developed a database schema consisting of 6 tables, which would later be expanded to 8 tables to account for users and images.

This schema accounts for the inventory, orders, customer details, and item details. It also links them with categories.

There is nothing overly complex here, but being able to link these tables using proper SQL to display the information either to the customer or the admin panel was an essential milestone.

Result

Reflection

Overstock was my first involvement playing a role as a developer using the SDLC within a group project, and I learned a ton over the course of 3 months. I learned the importance of each phase of the SDLC, how to communicate effectively within a group of multiple developers working on the same project, and how to utilize git as a group. We each played a vital role in getting the website to end product, and without each member it would not have looked as good as it did once complete. I'm very fortunate to have been able to work with each member of the group.

I think this website became more than we had anticpated it would become at the start. We began with the idea of a simple retail website that would not be intricate, but we couldn't help create a website that we would be proud to present. It has its limits, and it is not ready for any business to use.. However, we were able to creating a fully functioning website where users can add a product to cart and order the product. Admins are able to add, update, delete, and change sale dates on inventory. Admins are also able to add images to the database.

This website is one I'm proud to share.
