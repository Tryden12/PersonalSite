Bob's Pizza

Introduction

Bobs Pizza App is an application that allows users to create a custom pizza order. Items which are customizable include: type of pizza, toppings, drinks, tip, and delivery/pick-up. Users can view the menu and then place an order. On the order screen, the total is updated dynamically.

This app was the my first attempt at creating a professionally looking application. I worked solo on this project while attending APSU in the Spring of 2022.

Planning

The planning involved with this app took 7 days. As a beginner to Android Studio at the time of developing this app, I researched how to use widgts, intents, listeners, and various and how to use various resources such as drawables, strings, themes, colors, and launcher icons. I learned how to use the gradle correctly as well.

The plan for how the app would come together only took 2 of those 7 days, as I new from the beginning there would be four activities: Login, Create User, Home, and Order Screen. I researched how to print an order to a file in assets and how to dynamically update a TextView.

Result

Reflection

Bob's Pizza app was such an enjoyable ride developing. I learned so many of the basics, and gained loads of confidence for my skillset as a developer. There were many roadblocks along the way, but I was able to break through each of them to produce an app I am extremely proud of developing as my first Android app.

Looking back, there is a long, long list of things I would like to change, edit, include, and not include in the app. My dev skills have grown immensley since working on this app in February of 2022, so looking back on your first project is always fun. Hopefully one day I can revisit this app and make some improvements.
